CSS Transition Timing Functions

1. transition-timing-function: linear;

.ball {
    transition: left 3s;
    transition-timing-function: linear;
}

Hover over the box. The speed stays constant.

2. transition-timing-function: ease;

.ball {
    transition: left 3s;
    transition-timing-function: ease;
}

Hover over the box. Slow start, fast middle, slow end.

3. transition-timing-function: ease-in;

.ball {
    transition: left 3s;
    transition-timing-function: ease-in;
}

Hover over the box. Starts slowly and speeds up.

4. transition-timing-function: ease-out;

.ball {
    transition: left 3s;
    transition-timing-function: ease-out;
}

Hover over the box. Starts fast and slows down.

5. transition-timing-function: ease-in-out;

.ball {
    transition: left 3s;
    transition-timing-function: ease-in-out;
}

Hover over the box. Slow start, fast middle, slow end.
