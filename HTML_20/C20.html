Understanding CSS Transitions

1. Transition Property

.box{
    transition:
    background-color 1s;
}

.box:hover{
    background: crimson;
}

Hover the square to change its color smoothly.

What changes?

Only the background color changes.

2. Transition Duration

.box{
    transition:
    transform 3s;
}

Hover to move the box over 3 seconds.

Duration Timeline

Duration controls how long the animation takes.

3. Transition Timing Function

.box{
    transition:
    transform 2s
    ease-in-out;
}

Hover to see the movement speed change.

Ease-In-Out Graph

Slow start → Fast middle → Slow end.

4. Transition Delay

.box{
    transition:
    transform 2s;

    transition-delay:
    1s;
}

Hover and wait 1 second before movement starts.

Delay Timeline

The transition waits before starting.

Transition Shorthand

transition:
property duration timing-function delay;

/* Example */

transition:
transform 2s ease-in-out 1s;
